CSS (3n+3)
Using a formula (an + b). Description: a represents a cycle size, n is a counter (starts at 0), and b is an offset value.

Here, we specify a background color for all p elements whose index is a multiple of 3:

p:nth-child(3n+0) {
  background: red;
}
